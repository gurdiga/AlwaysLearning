Two ways to join the chat:

1. Javascript version is http://chat.daforest.org/
a. Click on "Advanced"
b. Choose server name -- Pegasus.mythnet.net -- from the drop-down
menu box
c. Choose channel name -- #Homeschool -- from the drop-down menu box
d. Type in your own unique nickname

Or

2. Using mIRC:
a. Open the Options box/Connect.
b. Choose the add button. Description -- MythNet
c. IRC server -- pegasus.mythnet.net -- and then click on Add. It
should now be in the drop down list to chose it.
d. Channel name -- #Homeschool. To get there, type /join
#Homeschool into the text box at the bottom of the status window and hit
enter. You will be taken to the homeschool chat room.
